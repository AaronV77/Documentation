
Here are the basic instructions on how to use GitHub from terminal!
Author: Aaron Valoroso

- From local / terminal
    - git config --global user.name "put your user name here"
    - git config --global user.email "put your email here"
    - git init my_repositiory (replace my_repositiory with a name you want)
    - cd my_repositiory
    - nano README (Put some mumbo gumbo into this document)
    - git add README
    - git add other files (git add all of your other files)
    - git commit -m "some message"
    - Go to Git hub's website and make a repository.
    - git remote add origin "URL"
    - git push origin master

- When you clone from git hub, and want to push
    - git clone "URL
    - git add --all
    - git commit -am "commit message"
    - git remote add origin "URL"
    - git push origin master

- When deleteing a local and remote branch
	- git branch -d "branch_name"
	- git branch --delete "branch_name"
	
- Merging a branch
	- git checkout "branch_name"
	- git merge master
	- git checkout master
	- git merge "branch_name"
